# Assets Folder

Place your ConnectHer brand assets here:

## Required Files:

1. **logo-horizontal.png** (or .svg)
   - Horizontal logo for navigation and footer
   - Recommended dimensions: 200px width (minimum height: 5mm as per brand guidelines)

2. **favicon.png** (or .ico)
   - Wordmark logo for favicon
   - Recommended dimensions: 32x32px or 64x64px
   - Should be square format

## Optional Files:

- Team headshots (if you want to add a team section)
- Additional illustrations or graphics from your brand guidelines
- Background images (if needed)

## Notes:

- All logo files should follow the brand guidelines
- Keep file sizes optimized for web (use compression if needed)
- SVG format is preferred for logos for scalability
